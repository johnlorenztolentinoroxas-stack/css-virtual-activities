Activity 02 · Hardware

Identify Power Supply Unit Components

Test your ability to recognize and label the internal components of a Power Supply Unit. This activity covers the transformer, rectifier, filter, and voltage regulator circuits and their roles in delivering stable power to the system.

⚡ Power Supply Unit 🔩 Hardware

Activity Overview

The Power Supply Unit is responsible for converting AC power from the wall outlet into stable DC voltages used by all components inside the computer. Understanding its internal parts is an essential CSS competency.

In this activity, you will be presented with a diagram or simulation of a PSU and asked to identify its components — transformer, rectifier, filter, and regulator circuits — along with their specific functions in the power conversion process.

Activity 03 · Software

Basic CMD Prompts

Practice using essential Windows Command Prompt commands through a simulated terminal environment. From file management and directory navigation to network diagnostics and system controls, this activity builds your command-line fluency.

💻 Command Prompt 🖱️ Software

Activity Overview

The Windows Command Prompt (CMD) is a powerful tool for managing files, diagnosing network issues, and performing system-level tasks. Proficiency in CMD is a core requirement in Computer System Servicing.

This activity will test your knowledge of 25 essential commands including dir, cd, ipconfig, ping, tasklist, chkdsk, and more. You will be given scenarios and asked to input or identify the correct command.

Activity 04 · Software

Making a Bootable USB

Walk through the complete process of creating a Windows 10 bootable USB flash drive using the Microsoft Media Creation Tool. This is a foundational skill for OS installation and system recovery in CSS.

💾 OS Installation Prep 🖱️ Software

Activity Overview

A bootable USB drive is the starting point for any operating system installation or repair. Creating one correctly — with the right language, architecture, and edition — is a critical step that precedes BIOS configuration and OS setup.

In this activity, you will simulate or sequence the 8-step process: downloading the Media Creation Tool, accepting the license, selecting media type, choosing system specifications, plugging in the USB, and waiting for the write to complete.

Activity 05 · Software

Configuring BIOS & Installing OS

Navigate BIOS settings and configure the boot order to install an operating system from a bootable USB. This activity covers the full 11-step process from entering BIOS to completing the Windows 10 installation.

⚙️ BIOS Configuration 🖱️ OS Setup

Activity Overview

BIOS (Basic Input/Output System) is the firmware that initializes hardware during startup. Configuring it correctly — disabling Fast BIOS Mode, setting AHCI to Manual, turning off Secure Boot, and selecting the right boot device — is essential before any OS installation can begin.

This activity will guide you through each BIOS screen and setting, then simulate the OS installation sequence that follows once the system boots from the USB drive.

Activity 06 · Software

Managing & Creating Partitions

Use the Windows Disk Management tool to shrink existing volumes, create new partitions, assign drive letters, and format storage using the NTFS file system. Proper partition management is essential for organized and efficient storage use.

🗂️ Storage Management 🖱️ Software

Activity Overview

Disk partitioning divides a physical storage drive into separate logical sections, each functioning independently. This allows for better file organization, dual-boot setups, and data separation between the OS and user files.

In this activity, you will simulate the 9-step partition creation process — opening Disk Management, shrinking an existing volume, creating a new simple volume, specifying its size and drive letter, and formatting it as NTFS. You will need to convert MB values correctly (1,024 MB = 1 GB).

Activity 07 · Security

Detecting & Removing Malware

Learn to recognize the warning signs of malware infection, and follow the complete removal process — from booting into Safe Mode and deleting temporary files, to resetting browser settings and running a full anti-malware scan.

🛡️ Malware Removal 🔐 Security

Activity Overview

Malware — malicious software designed to damage, disrupt, or gain unauthorized access to a system — is one of the most common problems encountered in computer servicing. Knowing how to detect it early and remove it safely is a core CSS competency.

This activity will present you with infected system scenarios and ask you to identify the signs of malware, then guide you through the correct removal steps: Safe Mode boot, Disk Cleanup, removing system restore points, resetting the browser, and scanning with an anti-malware tool.
